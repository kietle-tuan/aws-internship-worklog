## Final Project: AWS Stock Analyzer

The AWS Stock Analyzer project is a cloud-based stock analysis and alert system.
The system is designed to collect stock market data, process information, generate AI-assisted analysis, and display results through a dashboard.

The proposed architecture includes several AWS services, such as:

* Amazon S3
* Amazon CloudFront
* AWS WAF
* Amazon API Gateway
* Amazon Cognito
* AWS Lambda
* Amazon EventBridge
* Amazon SQS
* Amazon DynamoDB
* AWS KMS
* AWS Systems Manager Parameter Store
* Amazon Bedrock
* Amazon CloudWatch
* Amazon SNS

The architecture diagram is included in the proposal section.
